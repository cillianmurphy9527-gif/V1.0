# 🚀 如何重启服务器查看最新功能

## 问题说明

您看到的是旧版本界面，因为：
- ✅ 所有最新代码已经保存
- ❌ 但服务器还在运行旧的构建版本

## 📱 macOS 操作步骤（超详细）

### 第一步：打开终端

1. **按键盘上的 `Command + 空格`**（聚焦搜索）
2. **输入**：`终端` 或 `Terminal`
3. **按回车键**

现在您应该看到一个黑色或白色的窗口。

---

### 第二步：停止旧服务器

在终端中，**复制粘贴**下面这行命令，然后按回车：

```bash
pkill -f "next"
```

💡 **提示**：
- 复制：`Command + C`
- 粘贴到终端：`Command + V`
- 执行命令：按 `回车键`

这会停止所有正在运行的 Next.js 服务器。

---

### 第三步：进入项目文件夹

在终端中，**复制粘贴**下面这行命令，然后按回车：

```bash
cd /Users/liuyijia/agentcross
```

---

### 第四步：启动开发服务器

在终端中，**复制粘贴**下面这行命令，然后按回车：

```bash
npm run dev
```

---

### 第五步：等待启动完成

您会看到终端显示一些文字，等待大约 10-30 秒，直到看到类似这样的提示：

```
✓ Ready in 3.2s
○ Local:        http://localhost:3000
```

**看到这个提示后，就可以打开浏览器了！**

---

### 第六步：打开浏览器查看

1. **打开浏览器**（Chrome、Safari 等）
2. **在地址栏输入**：`http://localhost:3000`
3. **按回车键**

现在您应该能看到最新版本的网站了！🎉

## ❓ 常见问题

### Q1: 终端显示"command not found: npm"怎么办？

说明您的电脑没有安装 Node.js，需要先安装：

1. 访问：https://nodejs.org/
2. 下载并安装 LTS 版本
3. 安装完成后，重新打开终端
4. 再次执行上面的命令

---

### Q2: 端口 3000 被占用怎么办？

终端会显示类似 "Port 3000 is already in use" 的错误。

**解决方法**：使用其他端口

```bash
PORT=3001 npm run dev
```

然后访问：http://localhost:3001

---

### Q3: 如何停止服务器？

在运行服务器的终端窗口中：
- **按 `Control + C`**（注意是 Control，不是 Command）

---

### Q4: 还是看到旧界面怎么办？

1. **清除浏览器缓存**：
   - Chrome：`Command + Shift + Delete`
   - Safari：`Command + Option + E`

2. **强制刷新页面**：
   - 按 `Command + Shift + R`

3. **重新启动浏览器**
