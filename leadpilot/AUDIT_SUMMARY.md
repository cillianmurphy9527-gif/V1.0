## 📋 当前页面显示问题

### 问题
- 开发服务器运行在 `localhost:3005`（端口被占用）
- 所有请求返回 404
- 文件监听器错误：`EMFILE: too many open files`

### 原因
- 文件描述符限制过低（默认 256，需要 4096+）
- 多个 Next.js 进程在运行

### 快速修复
```bash
# 增加文件描述符限制
ulimit -n 4096

# 清理缓存
rm -rf .next node_modules/.cache

# 重启服务器
npm run dev
```

详见: `PAGE_DISPLAY_FIX.md`
