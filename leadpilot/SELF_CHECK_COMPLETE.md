1. **环境变量配置不完整**
   - 需要配置: DATABASE_URL, DEEPSEEK_API_KEY, GEMINI_API_KEY, RESEND_API_KEY, TURNSTILE_SECRET_KEY
   - 文件: `.env.local`

2. **短信验证码系统未实现**
   - 需要实现 Redis 存储和验证
   - 文件: `app/api/auth/send-code/route.ts`, `app/api/auth/register/route.ts`

3. **用户创建逻辑未实现**
   - 需要启用被注释的代码
   - 文件: `app/api/auth/register/route.ts`

4. **硬编码测试账号暴露**
   - 需要删除硬编码账号
   - 文件: `lib/auth.ts`

5. **数据库 Schema 与代码不同步**
   - 需要添加 `username` 字段
   - 文件: `prisma/schema.prisma`

6. **配额月度重置逻辑错误**
   - 需要修正日期计算
   - 文件: `lib/quota.ts`, `lib/feature-gate.ts`

7. **验证码泄露风险**
   - 生产环境不返回验证码
   - 文件: `app/api/auth/send-code/route.ts`

8. **缺少 CSRF 保护**
   - 需要实现 CSRF token 验证
   - 文件: 所有 POST/PUT/DELETE API 路由
